Ejemplo de Schema DBML

                
Crea un archivo .txt con este formato:

                
// Definir tablas
Table clientes {
  cliente_id integer [primary key]
  nombre varchar
  email varchar
  telefono varchar
}

Table pedidos {
  pedido_id integer [primary key]
  cliente_id integer
  fecha timestamp
  total decimal
}

// Definir relaciones
Ref: pedidos.cliente_id > clientes.cliente_id


                
Guía rápida:

                
    
                    
  • Tablas: Table nombre { ... }
    • 
                    
  • Columnas: nombre_columna tipo [opciones]
    • 
                    
  • Primary Key: [primary key]
    • 
                    
  • Relaciones: Ref: tabla1.col > tabla2.col
    • 
                    
  • Tipos comunes: integer, varchar, decimal, timestamp, boolean
